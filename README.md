# XCPlayer
基于 Qt6 和 FFmpeg 7.1.1 开发的本地播放器。

## 系统要求

  **操作系统**：**仅支持 Windows 10 及以上系统 (64-bit)**。

## 编译指南 (Windows)

  **本项目自带了 Windows 平台的 FFmpeg 7.1.1 开发库（Include & Lib）**。

  1. 安装 Qt 6.8 或更高版本。
  2. 使用 Qt Creator 打开 `CMakeLists.txt`。
  3. 直接点击构建（Build）即可。

  **关于 FFmpeg：**  
  `ffmpeg-7.1.1/` 为编译所需的头文件与导入库；  
  `ffmpeg-7.1.1/bin/` 目录为裁剪后的 FFmpeg 7.1.1 运行时动态库（DLL），已移除本项目不需要的组件，体积更小、加载更快。

## 主要功能
 1. 文件批量导入、文件夹扫描导入、lrc歌词文件批量导入。
 2. 支持自定义播放列表、自定义应用背景。
 3. 支持流切换、倍速切换、软硬解切换、HDR 切换、外挂字幕。
 4. 支持媒体信息解析、元数据修改。
 5. 支持桌面歌词（滚轮更改字体大小）、悬浮小窗口（滚轮修改音量）。
 6. 支持简单图像滤镜（VideoProcessor内置）
 7. 支持 WASAPI 独占模式。
 8. 支持硬件加速解码（D3D11VA）。
 9. 网络流播放（Beta）
 10. 简单的视频滤镜（d3d11videoprocessor 内置）
